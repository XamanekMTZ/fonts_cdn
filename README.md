# Fonts CDN

Repositorio de fuentes, utilizado como un CDN para proyectos web.

## Fuentes actualmente cargadas:

- Averia Serif Libre
- Crimson Text
- DM Sans
- JetBrains Mono
- Merriweather
- Momstype Writer
- Montserrat
- Nunito
- Plus Jakarta Sans

> [!NOTE]
> Para utilizarlas solo copia la URI correspondiente a cada hoja de estilos almacenada en el repositorio.
